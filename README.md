# La Puntilla — Landing Page

Página web profesional para **La Puntilla**, restaurante de mariscos ubicado en el Muelle Turístico de Mazatlán, Sinaloa.

## Tecnologías
- HTML5 semántico y accesible
- CSS3 modular (variables, @supports, legacy fallbacks)
- JavaScript vanilla modular (sin frameworks ni dependencias)

## Características
- Hero con efecto parallax multicapa y partículas animadas
- Galería fotográfica con lightbox
- Carousel de reseñas reales (táctil, con arrastre y auto-play)
- Efecto glow difuminado que sigue al cursor en la sección de platillos
- Reservaciones directas vía WhatsApp y teléfono
- Mapa de Google Maps embebido
- Soporte para navegadores antiguos con @supports y polyfills
- 100% responsive (móvil, tablet y desktop)

## Estructura
```
puntilla-fix/
├── index.html
├── img/          ← Fotos reales del restaurante
├── css/          ← 11 archivos CSS modulares
└── js/           ← 8 módulos JavaScript
```

## Uso
Abrir `index.html` directamente en el navegador. No requiere servidor ni dependencias.
